# Senior Android Testing Masterclass

This repository demonstrates how a Senior Android Engineer builds and maintains a robust testing pyramid across 15+ modules.

**Testing Stack:**
- Kotest for unit & property-based testing
- MockK + Turbine for ViewModel & Flow testing
- Jetpack Compose Testing for UI
- Macrobenchmark + Baseline Profile tests
- UI Automator for end-to-end flows

**Coverage Target:** 92%+ on domain and data layers.
